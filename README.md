# justmini

**justmini** is a React-based mini project that connects willing blood and organ donors with recipients. It provides search & matching, profile management, and secure contact flow for donations.

---

## Key Features
- Donor and recipient registration (with basic verification)
- Search donors by blood type, organ type, location, availability
- Admin dashboard to moderate listings and requests
- Responsive UI (mobile-first)
